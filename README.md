# Todo List 项目

一个使用多 Agent 协作开发的简单 Todo List 页面，用于演示和学习 Claude Code 多智能体协作开发流程。

## 项目简介

本项目是一个简单的 Todo List 网页应用，主要目标是验证和演示 **Claude Code 多智能体协作流程**。虽然项目功能简单，但完整展示了多个 Agent 如何协同工作完成开发任务。

### 多 Agent 协作模式

本项目采用以下 Agent 角色分工：

- **主 Agent (Coordinator)**: 负责任务调度、流程控制、决策协调
- **计划 Agent (Planner)**: 负责任务拆分、依赖分析、工作量评估
- **开发 Agent (Developer)**: 负责单个任务的独立开发
- **测试 Agent (Tester)**: 负责独立验收开发成果

### 协作流程

```
需求文档
   │
   ▼
计划 Agent ──► 任务清单
   │
   ▼
主 Agent ◄────────┐
   │              │
   ▼              │
开发 Agent        │
   │              │
   ▼              │
测试 Agent        │
   │              │
   ▼              │
测试通过?         │
   │         ┌───┴───┐
   │        是      否
   │         │       │
   │         ▼       └─────────┐
   │      下一个任务            │
   │                           │
   └───────────────────────────┘
       (回到原开发 Agent 修复)
```

## 如何运行项目

### 方法 1: 直接打开（推荐）

1. 找到项目根目录的 `index.html` 文件
2. 双击文件，使用默认浏览器打开
3. 开始使用 Todo List

### 方法 2: 使用本地服务器

```bash
# 使用 Python 3
python -m http.server 8000

# 使用 Python 2
python -m SimpleHTTPServer 8000

# 使用 Node.js (需要先安装 http-server)
npx http-server -p 8000
```

然后在浏览器中访问 `http://localhost:8000`

## 如何使用 Todo List

### 添加任务

1. 在输入框中输入任务内容
2. 点击"添加"按钮或按 Enter 键
3. 任务会出现在列表中

**示例**：
```
输入：学习 JavaScript
点击：添加按钮
结果：任务出现在列表中
```

### 删除任务

1. 找到要删除的任务
2. 点击任务右侧的 × 按钮
3. 任务从列表中移除

### 切换完成状态

1. 点击任务文本区域（不是删除按钮）
2. 任务会显示为已完成（删除线 + 变灰）
3. 再次点击可恢复为未完成状态

### 注意事项

- 不能添加空任务或纯空格任务
- 空输入时输入框会显示红色边框提示
- 按 Enter 键可以快速添加任务
- 列表为空时会显示友好提示
