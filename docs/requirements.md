# Todo List 页面需求文档

## 项目目标

开发一个简单的 Todo List 网页，用于学习 Claude Code 多智能体协作流程。

本项目不是为了做复杂功能，而是为了验证以下流程：

- 主 Agent 负责任务调度
- 计划 Agent 负责拆分任务
- 开发 Agent 负责单个任务开发
- 测试 Agent 负责独立验收
- 测试失败后回到原开发 Agent 修复
- 测试通过后进入下一个任务

## 技术要求

使用最简单的前端技术：

- HTML
- CSS
- JavaScript

暂时不要使用 React、Vue、后端服务或数据库。

## 功能需求

### 1. 页面结构

页面需要包含：

- 标题：Todo List
- 输入框
- 添加按钮
- 任务列表区域

### 2. 添加任务

用户在输入框中输入任务内容后，点击添加按钮，可以把任务添加到列表中。

### 3. 删除任务

每个任务旁边需要有删除按钮，点击后可以删除该任务。

### 4. 完成任务

每个任务可以点击切换完成状态。

完成后的任务需要有明显视觉区别，比如文字加删除线。

### 5. 空输入处理

如果用户没有输入内容，点击添加按钮时，不应该添加空任务。

## 最终交付文件

最终至少包含：

- `index.html`
- `style.css`
- `script.js`

## 初始限制

当前阶段只允许先搭建协作流程，不要一次性完成所有功能。
