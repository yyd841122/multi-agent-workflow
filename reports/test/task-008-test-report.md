    ```javascript
    function updateEmptyState() {
        if (taskList.children.length === 0) {
            emptyState.style.display = 'block';
        } else {
            emptyState.style.display = 'none';
        }
    }
    ```
  - 逻辑分析：
    - `taskList.children.length === 0` 表示没有任务
    - `taskList.children.length > 0` 表示有任务
    - 有任务时执行 `emptyState.style.display = 'none'`
  - 添加第一个任务后，`length` 从 0 变为 1，不满足 `=== 0` 的条件
  - 执行 `else` 分支，设置 `display = 'none'`
  - 空状态提示被隐藏
  - 逻辑正确，实现符合预期
