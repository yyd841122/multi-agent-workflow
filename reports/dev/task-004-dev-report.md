## 完成内容

### 修改的文件
1. **`script.js`** - 添加了完整的任务添加功能实现

### 实现的功能

#### 1. 添加任务函数
- 实现了 `addTask()` 函数,负责将输入框内容添加到任务列表
- 函数包含完整的任务创建流程:获取输入、创建元素、组装任务、添加到列表、清空输入、重新聚焦

#### 2. 事件绑定
- 为添加按钮绑定了点击事件监听器
- 使用 `addEventListener('click', addTask)` 实现点击响应

#### 3. 任务项结构
- 创建的列表项结构: `<li><span>任务文本</span><button class="delete-button">×</button></li>`
- 使用 `<span>` 包裹任务文本,为后续完成状态切换预留接口
- 使用 `<button>` 创建删除按钮,预留 `delete-button` class 样式

### 实现细节

#### 使用的主要 DOM API
- `document.getElementById()` - 获取页面元素(输入框、按钮、列表)
- `document.createElement()` - 创建新的 DOM 元素(li、span、button)
- `appendChild()` - 将子元素添加到父元素
- `textContent` - 设置和获取文本内容
- `value` - 获取和清空输入框内容
- `focus()` - 让输入框重新获得焦点

#### 元素创建和操作流程
1. 获取输入框的值(`taskInput.value`)
2. 创建 `<li>` 元素作为任务容器
3. 创建 `<span>` 元素存放任务文本
4. 创建 `<button>` 元素作为删除按钮
5. 将 span 和 button 添加到 li 中
6. 将 li 添加到任务列表 ul 中
7. 清空输入框(`taskInput.value = ''`)
8. 让输入框重新获得焦点(`taskInput.focus()`)

#### 用户体验优化
- 添加任务后自动清空输入框,方便连续添加
- 添加任务后自动聚焦输入框,无需手动点击即可继续输入
- 删除按钮使用 "×" 符号,简洁直观
