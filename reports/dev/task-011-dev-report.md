#### 4. 如何使用 Todo List
详细说明了三个核心功能的使用方法：

**添加任务**
1. 在输入框中输入任务内容
2. 点击"添加"按钮或按 Enter 键
3. 任务会出现在列表中

**删除任务**
1. 找到要删除的任务
2. 点击任务右侧的 × 按钮
3. 任务从列表中移除

**切换完成状态**
1. 点击任务文本区域
2. 任务显示为已完成（删除线 + 变灰）
3. 再次点击可恢复为未完成状态

**注意事项**
- 不能添加空任务或纯空格任务
- 空输入时输入框会显示红色边框提示
- 按 Enter 键可以快速添加任务
- 列表为空时会显示友好提示
