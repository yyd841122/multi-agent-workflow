### 实现细节

#### CSS 技术使用
1. **全局样式重置**：使用 `*` 选择器重置所有元素的 margin 和 padding，设置 `box-sizing: border-box`
2. **布局方式**：使用 `inline-block` 使输入框和按钮在一行排列
3. **响应式考虑**：使用百分比宽度 `calc(100% - 100px)` 确保输入框自适应
4. **交互效果**：使用 `:hover` 伪类和 `transition` 实现平滑的按钮效果
5. **焦点状态**：使用 `:focus` 伪类为输入框添加焦点提示

#### 配色方案
- 主色调：绿色 `#4CAF50`（按钮）
- 背景色：浅灰色 `#f5f5f5`（页面）、`#fafafa`（列表）
- 文字色：深灰色 `#333`（标题）、白色（按钮）
- 边框色：灰色 `#ddd`

#### 特殊处理
由于 HTML 结构中元素直接放在 body 下，没有包裹容器，因此使用 `body > *` 选择器来实现所有子元素的居中效果，既符合验收标准，又不需要修改 HTML 结构。
